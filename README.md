# Anthem Site

Static public site for Anthem, built for GitHub Pages.

Anthem is presented as an independent research lab and studio working across technology, land, biological intelligence, cryptographic coordination, AI-assisted code, and future forms of living.

## Structure

- `index.html` contains the site content.
- `styles.css` contains the Cargo-inspired editorial visual system.

## Publish on GitHub Pages

Open repository settings, go to Pages, and publish from the `main` branch root.

Add a custom domain later from the Pages settings.

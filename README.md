# Anthem Site

Static public site for Anthem, built for GitHub Pages.

Anthem is presented as an independent systems studio working across biology, design, software, physical coordination, material intelligence, ecological systems, and adaptive tools.

The site avoids research-journal, content-subscription, and SaaS positioning. The current commercial structure is Public / Licence / Consult.

## Structure

- `index.html` contains the homepage content.
- `support.html` contains the access and commercial structure.
- `styles.css` contains the editorial visual system.

## Publish on GitHub Pages

Open repository settings, go to Pages, and publish from the `main` branch root.

Add a custom domain later from the Pages settings.
